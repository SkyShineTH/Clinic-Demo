# Clinic Demo (`clinic-demo`)

สาธิตเว็บไซต์คลินิกทันตกรรมพรีเมียม (กรุงเทพฯ) + แอปพนักงานแบบจำลอง — **ไม่ใช่คลินิกจริง** ไม่มีข้อมูลคนไข้จริง (PHI)

## เริ่มต้น

```bash
npm install
npm run dev
```

เปิด [http://localhost:3000](http://localhost:3000)

## ฟีเจอร์หลัก

- **มาร์เก็ตติ้ง:** หน้าแรก (ไทย), บริการ, เทคโนโลยี, ทีม, แกลเลอรี่, สาขา, FAQ, หน้ากฎหมายร่าง, `/en` แบบย่อ
- **จองคิว:** `/booking` — ส่งคำขอไปที่ API ใน-memory (รีสตาร์ทเซิร์ฟเวอร์แล้วข้อมูลคำขอใหม่อาจหาย)
- **พนักงาน:** `/app/reception` แดชบอร์ดหน้าร้าน + เมนูไปยังหน้าอื่นที่เป็น stub

## ภาพประกอบ

- **Hero หน้าแรก:** `public/images/hero-clinic.jpg` — เครดิตใน `src/lib/hero-photo.ts` และใต้ภาพบนเว็บ
- **ชุดสต็อกเพิ่ม:** `public/images/points/*.jpg` — ดาวน์โหลดจาก Unsplash ใช้ในหน้าบริการ เทคโนโลยี ทีม แกลเลอรี่ สาขา FAQ จองคิว แดชบอร์ดพนักงาน ฯลฯ รายการ `alt` / ลิงก์หน้ารูปอยู่ใน `src/lib/stock-photos.ts` และลิงก์ย่อยใต้การ์ดหลายจุด
- **SVG เวกเตอร์:** `src/components/illustrations/`

## สแตก

- Next.js (App Router) · React · Tailwind CSS v4 · Noto Sans Thai

## หมายเหตุ

การจองและคิวจากเว็บเก็บในหน่วยความจำโปรเซสของเซิร์ฟเวอร์เพื่อเดโมเท่านั้น — ไม่เหมาะกับ production
